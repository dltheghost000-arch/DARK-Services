# DARK Services Tool

A simple Python tool that provides access to various services including:
- Websites
- Channels  
- Developer info
- Number bots (permission required)

## Requirements
- Python 3.x

## Installation
## 1_Packages Installation
```bash
pkg install python
pkg install git
```
## 2_Clone The Project
```bash
git clone https://github.com/dltheghost000-arch/DARK-Services.git
```
# Run the tool 
```bash
cd DARK-Services
python DARK.py
